# Readme for this archive
This archive contains a collection of files and resources related to ZZN second part of project. Below is a brief overview of the contents:
- `clustering.ipynb`: A Jupyter notebook file containing code, for clustering analysis and visualization.
- `final_*.py`: source code for data prediction models, split by method that was used for prediction.
- `zzn_proj.pdf`: A PDF document providing a detailed explanation of the project, including results, and conclusions.
- `requirements.txt`: A text file listing the Python packages and dependencies required
