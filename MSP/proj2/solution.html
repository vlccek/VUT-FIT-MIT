Weibullovo rozdlen

  • Weibullovo rozdlen je definovno dvma parametry:

Weibullovo rozdlen m hustotu pravdpodobnosti a distribun funkci definovanou nsledovn: $$ f(t; \lambda, k) = \frac{k}{\lambda} \left( \frac{t}{\lambda} \right)^{k-1} e^{-(t / \lambda)^k} $$ Funkce peit (SF) (pravdpodobnost peit do asu $S(t)$ je definovna jako: $$ S(t; \lambda, k) = e^{-(t / \lambda)^k} $$

Vpoet logaritmick vrohodnostn funkce

Vrohodnostn funkce a logaritmick vrohodnostn funkce pro Weibullovo rozdlen jsou definovny nsledovn:

$$ \mathcal{L}(k, \lambda) = \prod_{i} \left[ \frac{k}{\lambda} \left( \frac{x_i}{\lambda} \right)^{k-1} e^{-\left( \frac{x_i}{\lambda} \right)^k} \right] \prod_{j} \left[ e^{-\left( \frac{x_j}{\lambda} \right)^k} \right] $$

kde:

  • leny $\sum_i$ ($\prod_i$) zahrnuj pouze pozorovan selhn.
  • leny $\sum_j$ ($\prod_j$)zahrnuj pouze pravostrann cenzurovan body.

Logaritmick vrohodnostn funkce pro Weibullovo rozdlen:

$$ \ell(k, \lambda) = \sum_{i} \left[ \ln \frac{k}{\lambda} + (k - 1) \ln \frac{x_i}{\lambda} - \left( \frac{x_i}{\lambda} \right)^k \right] + \sum_{j} \left[ - \left( \frac{x_j}{\lambda} \right)^k \right] $$$$ \ell(k, \lambda) = n \ln(k) - n \ln(\lambda) + (k - 1) \sum_{i} \ln(\frac{x_i}{\lambda}) - \sum_{i} \left( \frac{x_i}{\lambda} \right)^k - \sum_{j} \left( \frac{x_j}{\lambda} \right)^k $$

Nalezen minima logaritmick vrohodnostn funkce

Nyn je poteba najt minimum pro nalezen MLE odhadu parametr $k$ a $\lambda$. V tomto ppad to udlme numericky pomoc optimalizanho algoritmu.

Derivace podle $k$

$$ \frac{\partial \ell}{\partial k} = \frac{n}{k} + \sum_{i} \ln(x_i) - \sum_{i} \left( \frac{x_i}{\lambda} \right)^k \ln\left( \frac{x_i}{\lambda} \right) - \sum_{j} \left( \frac{x_j}{\lambda} \right)^k \ln\left( \frac{x_j}{\lambda} \right) $$

Derivace podle $\lambda$

$$ \frac{\partial \ell}{\partial \lambda} = -\frac{n}{\lambda} + k \sum_{i} \left( \frac{x_i}{\lambda} \right)^k \frac{1}{\lambda} + k \sum_{j} \left( \frac{x_j}{\lambda} \right)^k \frac{1}{\lambda} $$

import pandas as pd
import numpy as np
from scipy.optimize import minimize

failures = data['time'][data['censored'] == 0]
censored_times = data['time'][data['censored'] == 1]


# Log-likelihood function
def llf(n, k, lambd, x_values, y_values):
    term1 = n * np.log(k)
    term2 = -n * k * np.log(lambd)

    term3 = (k - 1) * np.sum(np.log(x_values))

    term4 = -np.sum((x_values / lambd) ** k)

    term5 = -np.sum((y_values / lambd) ** k)
    return term1 + term2 + term3 + term4 + term5


# estimate initial parameters
initial_params = [0.5, np.median(data['time'])]

# Optimalizace
result = minimize(
    fun=lambda params, f, c: -llf(len(f), params[0], params[1], f, c),
    # minus bcs minimizing, f-failrues, c-censored, len(f)-n from the formula
    x0=initial_params,
    args=(failures, censored_times),
    method='L-BFGS-B',  # from docs
    bounds=[(0.001, None), (0.001, None)]  # Lower bounds for k and : k > 0,  > 0
)

# Results:
k_hat, lam_hat = result.x
print("MLE odhad parametr pomoc scipy.optimize.minimize:")
print(f"Optimal shape parameter k: {k_hat:.3f}")
print(f"Optimal scale parameter : {lam_hat:.3f}")

Je exponenciln rozdlen je postaujcm modelem zapsanch dat?

$$ H_0: \text{Model je exponenciln ($k=1$)}\quad $$$$ H_1: \text{Model je Weibullovo ($k \neq 1$, nen exponenciln)} $$

MLE pro exponenciln rozdlen

Pro necenzurovan data

$$ L(\lambda) = \prod_{i=1}^n \lambda e^{-\lambda x_i} $$$$ \ell(\lambda) = \ln L(\lambda) = n \ln \lambda - \lambda \sum_{i=1}^n x_i $$

Pro cenzurovan data

$$ L(\lambda) = \prod_{i=1}^n e^{-\lambda x_i} $$$$ \ell(\lambda) = \ln L(\lambda) = - \lambda \sum_{i=1}^n x_i $$

Testovna hypotza

$$ \Lambda = -2 \ln \left(\frac{L(H_0)}{L(H_1)}\right) = 2[\ln L(H_0) - \ln L(H_1)]. $$$$ p = P(\chi^2(1) \geq \Lambda) = 1 - F_{\chi^2(1)}(\Lambda), $$

from scipy.stats import chi2


def exponential_log_likelihood_censored(lambda_, censored):
    logL_censored = -lambda_ * np.sum(censored)

    return logL_censored


def exponential_log_likelihood_failed(lambda_, f):
    logL_failures = np.sum(np.log(lambda_) - lambda_ * f)

    return logL_failures


def exponential_log_likelihood(lambda_, f, c):
    return -(exponential_log_likelihood_failed(lambda_, f) + exponential_log_likelihood_censored(lambda_, c))


initial_params = [1.]

result = minimize(exponential_log_likelihood,
                  x0=initial_params,
                  args=(failures, censored_times),
                  method='L-BFGS-B',  # from docs
                  bounds=[(0.001, None)])

lambda_head_exp_num_both = result.x[0]
# Results:
ll_weibull = llf(len(failures), k_hat, lam_hat, failures, censored_times)
ll_exp = -exponential_log_likelihood(lambda_head_exp_num_both, failures, censored_times)

LR = 2 * (ll_weibull - ll_exp)

Vybran body pro odhad

Stedn hodnota:

$$ E[T] = \lambda \cdot \Gamma\left(1 + \frac{1}{k}\right) $$

10% Percentil:

$$ P(T \leq t) = 0.1 $$

Vyjdeme ze vztahu pro distribun funkci: $S(t) = 1 - e^{-(t/\lambda)^k}$ a vyeme pro $t$:

$$ t = \lambda \cdot (-\ln(1 - p))^{1/k} $$$$ t = 7.429 \cdot (-\ln(1 - 0.1))^{1/6.173} $$$$ t = 5.16 $$
